# TrackBarLab — Лабораторная работа №2  
**Дисциплина:** Проектирование программного интерфейса  
**Вариант 16**

Простое Windows Forms-приложение на C#, демонстрирующее работу с компонентом `TrackBar` и изменение его свойств через модальное диалоговое окно.

### Что реализовано
- Главная форма с `TrackBar` и кнопкой «Настройки»
- Модальное окно для изменения:
  - `Minimum`, `Maximum`, `Value`
  - `Orientation` (горизонтальная / вертикальная)
  - `TickFrequency`
- Двусторонний обмен данными через свойства (get/set)
- Автоматическая адаптация размеров и положения главной формы при смене ориентации
- Все окна по центру экрана, фиксированный размер, без кнопки «Развернуть»
- Чистый код, осмысленные имена, комментарии
